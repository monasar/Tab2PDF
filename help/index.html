Getting Started!

1. First start by clicking the “Open” button, highlighted in the diagram below. “Save” and “Print” will be disabled until a file is opened.

2.  A file chooser should appear, navigate to the desired file and open it by either double clicking or highlighting it and clicking “Open”, and example of a file chooser is shown below with “moonlightsonata.txt” being the converted file.

3. Once a file is selected the windows will expand revealing a preview of the converted text file (red box), control for changing the view (blue box), and controls to edit the format, title and author of the PDF (green box). If you do not want any changes skip to step 4.

3a. The preview gives a look at the PDF before saving, in this example Moonlight Sonata is previewed

3b. If you want a closer look at the PDF, zoom-in by dragging the slider to the right, and zoom-out by  dragging the slider to the left.

3c. To alter the PDF change the title, title font size, author, author font size, font type, spacing of bars, staffs and lines.

4. To save the PDF select the “Save” button, highlighted in the diagram below.

5. A file chooser will appear allowing you to select what directory to save in (blue box) and give the PDF a name (red box), in this example since Moonlight Sonata is being saved by default the file is given the name “Moonlight Sonata.pdf” but may be changed to your liking.
